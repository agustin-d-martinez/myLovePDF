# myLovePDF
 App de manejo de PDFs y Archivos. Creada con Python y ayuda de QTDesign.
 
## Lenguaje:
- Python

## Programas:
- Visual Studio Code.
- QTCreator.

## Funcionalidad:
 Posee herramientas de manejo de PDF, imágenes, archivos multimedia y archivos en general.
 
 ### PDF:
 - Unir PDF: Une los PDF seleccionados en uno solo.
 - Separar PDF: Separa un PDF en otro solo con las hojas seleccionadas.
 - Separar Hojas: Separa las hojas seleccionadas de un PDF en archivos individuales.
 - Hoja en blanco: Agrega hojas en blanco al PDF.

 ### Imágenes:
 - Imagen a PDF: Trasnforma la imagen/es en un PDF de tamaño A4.
 - PDF a imagen: Transforma el PDF en multiples imágenes (1 por hoja).
 - Extensión de imagen: Cambia la extensión de un archivo de imagen.

 ### Archivos:
 - Renombrar: Renombra todos los archivos seleccionados siguiendo un template.

 ### Multimedia:
 - Tags Música: Coloca los tags a un archivo MP3 seleccionado.
 - MP4 a MP3: Transforma un archivo de video a uno de audio.
 - Descargar Música: Descarga el video de Youtube.
